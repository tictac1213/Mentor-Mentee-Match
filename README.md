## Setup Instructions

### Prerequisites
- Node.js (v16+)
- MySQL (v8+)
- Git

### Installation
1. Clone repository:
   ```bash
   git clone https://github.com/kk1701/mentorship-matching
   cd mentorship-matching
2. Install Dependencies:
    ```bash
    npm install
3. Create .env file:
    ```bash
    PORT=3000
    DB_HOST=localhost
    DB_USER=your_username
    DB_PASSWORD=your_password
    JWT_SECRET=your_random_secret_key
4. Start server:
    ```bash
    npm start

### Features
